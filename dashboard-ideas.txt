================================================================================
                        ADMIN DASHBOARD FEATURE IDEAS
================================================================================
Based on superadmin privileges (schema-policies.js) + database schema (schema.sql)

--------------------------------------------------------------------------------
1. GOD MODE USER PANEL
--------------------------------------------------------------------------------
   - Full User Dossier: Click any user -> see everything (images, captions,
     likes, saves, shares, screenshots, reports filed, reports against them,
     university, dorm, major)
   - User Timeline: Chronological activity feed of everything they've done
   - Ghost Mode: View the app exactly as that user sees it
   - Account Takeover: Edit their profile, change name/email, toggle admin flags
   - Nuke Button: One-click cascade delete (user + all their content)

--------------------------------------------------------------------------------
2. CONTENT SURVEILLANCE CENTER
--------------------------------------------------------------------------------
   - Private Caption Voyeur: Browse all private captions users thought nobody
     could see
   - Image X-Ray: See every image with AI-generated image_description and
     celebrity_recognition data
   - Embedding Explorer: 2D/3D visualization of image embeddings, find similar
     images
   - Caption Plagiarism Detector: Find duplicate/similar captions across users
   - Viral Predictor: Rank captions by like_count + share + screenshot velocity

--------------------------------------------------------------------------------
3. MODERATION WAR ROOM
--------------------------------------------------------------------------------
   - Report Triage Queue: Unified view of reported images + captions with reasons
   - One-Click Justice:
       * Dismiss (delete report, keep content)
       * Warn (flag user)
       * Remove (delete content + report)
       * Ban (delete user entirely)
   - Repeat Offender Tracker: Users with multiple reports against them
   - False Reporter Detection: Users who file frivolous reports

--------------------------------------------------------------------------------
4. AI/LLM CONTROL ROOM
--------------------------------------------------------------------------------
   - Prompt Playground: Edit humor_flavor_steps prompts live, test variations
   - Response Autopsy: View raw llm_model_responses with processing times
   - Chain Debugger: Visualize multi-step llm_prompt_chains flow
   - Model Showdown: Compare outputs across different llm_models for same image
   - Temperature Tuner: A/B test different llm_temperature settings
   - Caption Examples Curator: Manage few-shot examples for training

--------------------------------------------------------------------------------
5. HUMOR LABORATORY
--------------------------------------------------------------------------------
   - Flavor Factory: Create/edit humor_flavors with custom prompt chains
   - Theme Mixer: Map flavors to humor_themes (sarcasm, absurdist, etc.)
   - Flavor Leaderboard: Which humor styles get most engagement
   - Personality Workshop: Define personalities for transcript-based training
   - Slang Dictionary: Manage terms with definitions and examples

--------------------------------------------------------------------------------
6. RESEARCH COMMAND CENTER
--------------------------------------------------------------------------------
   - Study Builder: Create studies with date ranges, image sets
   - Participant Manager: Bulk toggle is_in_study, filter by criteria
   - Data Exporter: Export study captions, votes, engagement for analysis
   - A/B Test Dashboard: Compare humor_flavor_mix performance
   - Vote Analyzer: Deep dive into caption_votes patterns

--------------------------------------------------------------------------------
7. UNIVERSITY INTELLIGENCE
--------------------------------------------------------------------------------
   - Campus Heatmap: Engagement by university, drill down to dorms
   - Major Breakdown: Which majors are funniest (by caption engagement)
   - Dorm Wars Leaderboard: Pit dorms against each other
   - Domain Gatekeeper: Manage allowed_signup_domains (.edu allowlist)
   - Invitation Funnel: Track invitations conversion rates by school

--------------------------------------------------------------------------------
8. VIRALITY & ENGAGEMENT ANALYTICS
--------------------------------------------------------------------------------
   - Share Destination Breakdown: Where content goes (via share_to_destinations)
   - Screenshot Heatmap: Most screenshotted captions (pre-viral detection)
   - Like Velocity Graph: Real-time like accumulation curves
   - Save-to-Share Ratio: caption_saved vs shares analysis
   - Peak Hours: When users are most active by timezone

--------------------------------------------------------------------------------
9. COMMUNITY CONTEXT ENGINE
--------------------------------------------------------------------------------
   - News Injector: Manage news_snippets for topical caption generation
   - Entity Extractor View: See parsed news_entities (people, orgs, events)
   - Context Scheduler: Time-bound community_contexts with priority
   - Tag Manager: Organize contexts with community_context_tags
   - Embedding Search: Find relevant contexts by semantic similarity

--------------------------------------------------------------------------------
10. LINK & GROWTH TOOLS
--------------------------------------------------------------------------------
   - Short Link Factory: Create link_redirects with tracking
   - Click Analytics: visit_count trends over time
   - Campaign Folders: Organize links by folder_path
   - QR Generator: Generate QR codes for redirect links

--------------------------------------------------------------------------------
11. ERROR & BUG OBSERVATORY
--------------------------------------------------------------------------------
   - Bug Report Inbox: Triage user-submitted bugs with subject/message
   - TestFlight Crash Log: Monitor testflight_errors from iOS beta
   - Error Patterns: Cluster similar errors, identify systemic issues

--------------------------------------------------------------------------------
12. DANGEROUS POWERS PANEL
--------------------------------------------------------------------------------
   - Mass Caption Editor: Bulk edit captions (toggle is_public, is_featured)
   - Image Ownership Transfer: Reassign images between users
   - Force Publish: Make private captions public without user consent
   - Fake Engagement: Manually inflate like_count for promotion
   - Backdated Content: Modify created_datetime_utc timestamps
   - Shadow Delete: Remove content without notification

--------------------------------------------------------------------------------
13. MATRIX ADMIN TOOLS (for is_matrix_admin users)
--------------------------------------------------------------------------------
   - Matrix User Manager: Separate admin tier management
   - Permission Escalation Log: Track who granted admin to whom

--------------------------------------------------------------------------------
14. DATA EXPORT & BACKUP
--------------------------------------------------------------------------------
   - Full Database Dump: Export everything for backup/analysis
   - User Data Package: GDPR-style export for any user
   - Study Dataset Builder: Custom exports for research papers
